# Local development harness

These files let you run the example in a plain browser, with the keyboard
standing in for the controller — they are **not** part of what you ship.

- `index.html` — local test page (controls help + scoreboard overlay)
- `mock-platform.js` — local test harness + keyboard inputs

For how to run it (it must be served over **http**, not `file://`), the keyboard
controls, and what it stands in for, see
[**Local development** in the main README](../README.md#local-development).
